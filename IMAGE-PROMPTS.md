# Quikle Website v6 - Image Prompts

**Brand Identity:**
- **Theme:** "Your time back, your growth forward... Quickly"
- **Style:** High-end tech, glassmorphism, clean UI interfaces, futuristic but grounded.
- **Palette:** Midnight Blue (#191970), Teal (#008080), Cyan (#00F2FF), Coral accents (#FF5F5F).
- **Vibe:** Professional, efficient, automated, calm.

## 1. Hero Dashboard (hero-dashboard.jpg)
**Prompt:**
A futuristic, glassmorphic dashboard interface floating in a clean, abstract 3D space. The dashboard displays "Time Saved: 20hrs" and "Revenue: +20x" with glowing teal graphs and data visualizations. The background is a soft gradient of deep midnight blue and teal. High-tech, sleek, Apple-like aesthetic. 8k resolution, photorealistic 3D render.
